# 💕 Love Proposal - TEA LIFE 💕

A fun, romantic love proposal website built for B.Tech students to express their feelings with 3D animations and interactive elements!

## 🌟 Features

✨ **Photo Upload** - Upload your favorite photo
👤 **Personal Info** - Enter your name and confirm if you're AKKI
👯 **Friend Selection** - Choose your best friend (HINDHU, mouni, yogi, akki, or MR.BHANU)
⏱️ **Duration Selection** - Select friendship duration (1 month, 2 years, 3 years, 24 months)
🎨 **3D Animation** - Beautiful 3D animated hearts and tea emoji with realistic rendering
💌 **Romantic Messages** - Personalized love proposal messages
📱 **Share** - Share your love proposal with friends

## 📋 How It Works

1. **Upload Photo** - Start by uploading your favorite photo
2. **Enter Name** - Tell us your name
3. **Confirm Identity** - Confirm if your name is AKKI
4. **Pick Best Friend** - Select your best friend from the list
   - Clicking on friends (except MR.BHANU) shows an arrow animation
   - Clicking MR.BHANU shows "PERMISSION GRANTED" message
5. **Select Duration** - Choose how long you've been friends
6. **View 3D Proposal** - See the beautiful 3D animated proposal with personalized message
7. **Share** - Share your love proposal with others

## 🎯 Unique Features

- 🎬 **3D Animations** - Three.js powered realistic 3D scene with rotating hearts and floating tea emoji
- 🎨 **Beautiful UI** - Modern gradient design with smooth transitions
- 💌 **Romantic Messages** - Special "TEA LIFE" themed messages
- 📱 **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- 🔄 **Friend Shortcodes** - Each friend has a unique shortcode (H, M, Y, A, B)

## 🛠️ Tech Stack

- HTML5
- CSS3 (with animations)
- JavaScript (Vanilla)
- Three.js (for 3D graphics)

## 🚀 How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Follow the steps to create your love proposal
4. Share with your special someone! 💕

## 📁 Files

- `index.html` - Main HTML structure

## 💡 Customization

You can customize:
- Friend names in the `friends` array
- Duration options in the `durations` array
- Colors in CSS variables
- 3D animation parameters in `create3DAnimation()` function
- Messages in the final screen
